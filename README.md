SwiftBackend project
=======================

Server Side Swift using Kitura, Kitura CouchDB, Helium Logger and SwiftyJSON

## Getting Started

cd the project
run swift build
run .build/debug/SwiftBackend

or

open the SwiftBackend.xcodeproj with Xcode and simply run
